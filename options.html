Welcome to the SearchPin Extension for Chrome!

To perform any of the commands in SearchPin, start by entering > into your omnibox (the URL bar at the top of the browser) and pressing Space or Tab.

SearchPin can be used to help speed up searching for things. For instance, if you want to search for a number of things at once, try > cats | dogs to run two searches.

Or you could open a search in a background tab by ending your search with &.

For a full list of SearchPin features, go to the features page. For help on a specific command, go to the reference page.

Basic Searching Usage

> [search terms]

 Description

Entering search terms will start a search for those terms.

 Example

> cat pictures

This will search for pictures of cats.

 Prefixes Usage

> +[prefix]

> -

 Description

Setting your prefix will make SearchPin add that prefix to all searches you make.

> - will clear your prefix

 Example

> +javascript

This will set your prefix to javascript, which means you can search for relevant things without retyping it each time.

> arrays

> tips

This will search for "javascript arrays" and "javascript tips".

 Multisearch Usage

> [search 1] | [search 2] | ...

 Description

Multisearches will perform many different searches in with command. These searches will also use your prefix.

 Example

> +london

> bus times | films | restaurants

 Suffixes Usage

> [suffix] > [shortcut]

> (search) ![shortcut]

> -[shortcut]

 Description

Suffixes are shortcuts to add longer parts of searches quickly. When you make a suffix, you assign it a shortcut which you can use to add the suffix to your search again.

These can be used to store long search features you might want to use in the future, or to store parts of searches you use often.

 Example

> filetype:pdf > pdf

> books !pdf

> definition > !d

> !d nifty

 Search Targets Usage

> (search) @[target]

 Description

Search targets can be used to search in different places. Search targets can be viewed and edited in the options menu. Search targets also work with prefixes and multisearches.

 Example

> cats @flickr

> cats @yt

> searchpin @git
Reference
Search > search terms Enter terms to perform a search on your default target.
Multisearch > search 1 | search 2 Enter searches separated by | to perform many searches at once. Each search will open in a separate tab.
Options > options Opens the help and options pages.
Targetting > (command) @[target] Add @target to your search to perform a search on that target instead.
Set prefix > +[prefix] Sets your prefix, which will be added to all subsequent searches and multisearches.
Clear prefix > - Clears your prefix.
Use suffix > (search) ![shortcut] Adds the corresponding suffix to your search.
Add new suffix > [suffix] > [shortcut] Creates a new suffix.
Remove suffix > -[shortcut] Removes a suffix.
Background Search > (search) & Opens the search in a new tab.
Search targets

Here you can set up your search targets. These let you search on different websites. To use a search target, write the shortcut at the end of your search.

For example, to search for cats on Google, you would use > cats @google

Your default target will be used when you don't specify any target.
